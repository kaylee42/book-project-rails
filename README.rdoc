== README

Books for Nerds is a social Rails app for rating and recommending books. 

It's built in Ruby 2.2.3, and the Gemfile includes all necessary dependencies! Feel free to fork and experiment as much as you'd like. The program includes a database set-up and mulitple model associations, including the relationship that allows for friendships, and are easily customizable to operate in either the Facebook or Twitter model. 

Please reach out with any questions or feedback.
